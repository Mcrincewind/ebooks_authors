Ένα πρόγραμμα βιβλιοθήκης, στο οποίο μπορείς να προσθέσεις βιβλία, να βάλεις συγγραφείς και να ορίσεις ποιο βιβλίο ανήκει σε ποιον συγγραφέα, καθώς και να σου τα εμφανίσει. Το πρόγραμμα έχει υλοποιηθεί σε Java με Spring Boot.

Το frontend έχει δύο εκδόσεις: η μία βρίσκεται στο φάκελο frontend και είναι με απλή JavaScript, και η άλλη στο φάκελο ebooks-manager με React.

Προσοχή: Για να τρέξει το πρόγραμμα χρειάζεται η σωστή δομή της βάσης δεδομένων, η οποία βρίσκεται σε αρχείο με όνομα Data_Base. Οποιαδήποτε αλλαγή πρέπει να γίνει με προσοχή ώστε να μην επηρεαστεί η λειτουργία του προγράμματος.


//

This is a library management program where you can add books, assign authors, and link which book belongs to which author, and display all this information. The program is developed in Java using Spring Boot.

The frontend has two versions: one located in the frontend folder built with plain JavaScript, and another in the ebooks-manager folder built with React.

Important: To run the program correctly, you need the proper database structure, which is provided in a file named Data_Base. Any changes must be made carefully to ensure the program continues to function properly.
